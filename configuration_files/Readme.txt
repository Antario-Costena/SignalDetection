Folder with yolo configuration files.
For training, during configuration, rename the file yolo-obj_Training.cfg to yolo-obj.cfg and proceed with uploading the configuration files.
For detection, during configuration, rename the file yolo-obj_Detection.cfg to yolo-obj.cfg and proceed with uploading the configuration files.
In this phase it is necessary to increase the network resolution in the .cfg file (height = 832, width = 832 or any multiple of 32) - it will increase the precision.
